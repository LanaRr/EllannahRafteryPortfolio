E-commerce Website

HTML CSS Bootstrap and PHP

This project is an E-Commerce website that I built for my Web Applications module in college. The requirements of this projects are as follows

  • Must use the bootstrap framework
  • Include an image carosel
  • Links to a MySql database
  • Feature a login form
  • User must be able to add products to a cart and purchse items

Sweet Shack

The website I designed above is called Sweet Shack. An online sweet shop that sells a variety of sweets and treats. To keep to the theme of candy I designed the key elements with red and white with a lighter opaque striped background.

The functionality of this website contains some Bootstrap elements which are styled with HTML and CSS. The home page contains a carousel which displays an array of images. To display the products, I create a database that stores all the information of the product including an image. To display all the products, I use PHP to connect to the database and return the data in a while loop. PHP parses all the information to the website which then is styled into bootstrap cards. I used this method instead of hard coding the products so that if there is a change that needs to be made or a product to be added it can be easily fixed by amending the database.

There is a simple login feature implemented using a bootstrap form. This checks to make sure that a valid email address has been entered. For now, the password is hard coded in. I intend to create a database to store these details as to add extra functionality.
